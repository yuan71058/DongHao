# 项目截图文件夹

此文件夹用于存放项目案例的截图。

## 需要的图片文件

请将项目截图放入此文件夹，文件命名如下：

| 文件名 | 对应项目 |
|--------|----------|
| `project-ai-assistant.jpg` | 冬浩AI办公助手 |
| `project-auth-system.jpg` | 炸鸡网络验证系统 |
| `project-device-control.jpg` | 星辰中控 |

## 图片建议

- 格式：JPG 或 PNG
- 尺寸：建议 800x600 或 1200x800
- 大小：单张图片建议不超过 500KB
- 内容：展示项目主要界面或功能亮点

## 使用方法

将截图文件放入此文件夹后，网站会自动加载显示。
